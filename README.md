## 커밋 메시지 규칙
| 타입  | 내용           |
|-----|--------------|
| Add | 파일 추가에 대한 커밋 |
| Modify | 파일 수정에 대한 커밋 |
| Delete | 파일 삭제에 대한 커밋 |

- 예시
  - git commit -m "Add: 파일명(ex. 플랫폼_문제번호)"
